ROBO-Pet
ROBO-Pet is a fun and interactive Tamagotchi-style pet that you can care for directly within VS Code! Watch as your robot pet's happiness, age, and state change over time, and take care of it by feeding and interacting with it.

Features:
- Updates your pets happiness based on how much you code.
- You can change your robot's name to whatever you like.
- Watch your robot change emotions based on its happiness levels.
- Keep your pet as happy for as long as possible.

Installation:
1. Open Visual Studio Code OR follow this link.
2. If in Visual Studio Code, go to EXTENSION tab and search 'ROBO-Pet'.
3. Click "Install".

Women in Computer Science (WiCS) is a club at Arizona State University and they host a Hackathon every year to all genders. I did the 2024-2025 Spring Hackaton where I made a VS code extension to ultimately help programmers code more. With this goal in mind I ended up create a Tamagotchi-style clone in VS code where you take care of a little robot and to take care of said robot you have to code to make it happy.
